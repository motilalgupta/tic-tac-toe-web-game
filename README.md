# Tic-Tac-Toe Web Game

A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The game allows two players to take turns marking X and O on a 3×3 grid until a player wins or the game ends in a draw.

## Features

* Interactive 3×3 game board
* Two-player gameplay (X vs O)
* Winner detection logic
* Draw detection
* Game reset functionality
* Responsive and user-friendly interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Open the project folder.

3. Run `index.html` in your browser.
